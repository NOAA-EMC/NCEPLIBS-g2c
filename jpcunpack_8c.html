int dec_jpeg2000 ( char *  injpc,
g2int  bufsize,
g2int *  outfld 
)

This Function decodes a JPEG2000 code stream specified in the JPEG2000 Part-1 standard (i.e., ISO/IEC 15444-1) using JasPer Software version 1.500.4 (or 1.700.2) written by the University of British Columbia and Image Power Inc, and others.

JasPer is available at http://www.ece.uvic.ca/~mdadams/jasper/.

Parameters
injpcInput JPEG2000 code stream.
bufsizeLength (in bytes) of the input JPEG2000 code stream.
outfldOutput matrix of grayscale image values.
Returns
  • 0 Successful decode
  • -3 Error decode jpeg2000 code stream.
  • -5 decoded image had multiple color components. Only grayscale is expected.
Note
Requires JasPer Software version 1.500.4 or 1.700.2.
